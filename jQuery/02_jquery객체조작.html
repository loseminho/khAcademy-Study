3. attr(), removeAttr() : 엘리먼트 속성 값 확인/수정, 삭제하는 메소드

4. css() : 선택된 요소의 속성 값을 가져오거나 설정하는 메소드

5. html(), text(), val()

html() : 선택된 요소의 컨텐츠영역(innerHTML)을 가져오거나 수정

text() : 선택된 요소의 컨텐츠영역(innerText)을 가져오거나 수정 -> 컨텐츠 영역 중 태그는 안가져옴

val() : 선택된 요소의 value를 가져오거나 수정
