효과

페이지에 애니메이션 효과를 만들기 위한 메소드

사용방법 : $("선택자").메소드([speed],[easing],[callback]);

speed : 실행속도(단위 : ms)

callback : 효과 메소드 수행 후 수행할 함수 지정

easing : 변경되는 지점 별 속도(linear : 일정하게 | swing : 처음과 끝이 약간 느리게)

단, easing을 사용하기 위해서는 일반 jquery가 아닌 jquery-ui라이브러리를 추가로 사용해야 함.

animate() : 현재 css 속성을 설정한 값으로 차츰 변경되는 효과를 주는 메소드

$("선택자").animate({변경할 css 속성},[speed],[easing],[callback]);

단, easing을 사용하기 위해서는 일반 jquery가 아닌 jquery-ui라이브러리를 추가로 사용해야 함.

모든 속성이 적용되는 것은 아니며, jquery-ui를 사용하면 더 많은 속성 적용 가능(ex.background-color는 안됨)
