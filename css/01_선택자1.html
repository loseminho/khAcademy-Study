CSS 선택자

선택자
특정한 HTML 태그를 선택할 때 사용하는 기능
태그를 선택하여 원하는 스타일과 기능 적용 가능

전체 선택자

HTML 문서 안의 모든 태그를 선택 할 때 사용하는 선택자

태그 선택자

HTML 문서 안의 같은 태그를 모두 선택할 때 사용 

p { color : blue;}

아이디 선택자

HTML 문서 내 해당하는 아이디 속성을 가진 태그만 선택할 때 사용 (아이디는 중복 사용 X) 

        #아이디명{color : blue;}

        .클래스명{color : blue;}
