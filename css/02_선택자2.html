자손 선택자/ 후손 선택자

자손선택자 : 바로 아래 하위요소 // 후손 : 하위 요소 전체 

        자손 선택자 : 선택자>자손선택자{color:red;}
        후손 선택자 : 선택자 후손선택자{color:blue;}

동위선택자

동위 관계에서 뒤에 있는 태그를 선택할 때 사용 

        바로 다음요소 선택 : 선택자a + 선택자b {color:red;}
        다음 모든 태그 : 선택자a ~ 선택자b {color:red;}

반응선택자

사용자의 움직임에 따라서 달라지는 선택자 

        사용자가 클릭한 태그 선택자 : active{color:red;}
        사용자의 마우스가 올라간 태그 선택자 : hover{color:red;}
