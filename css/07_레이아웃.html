테두리 스타일

border-style : 테두리 스타일 지정

선택자{border-[방향]-style : none | hidden | dashed | dotted | double | groove | inset  | outset | ridge | solid}

border-width : 테두리 두께

선택자{border-[방향]-장소 : thin | medium | thick | px }

border-color : 테두리 색상

선택자{border-[방향]-color : 색상표현;}

border : 테두리, 스타일, 두께, 색상을 한번에 지정

선택자{border-[방향] : width style color;}

border-radius : 박스 모서리를 둥글게 하는 속성

선택자{border-[방향]-radius: 숫자px;}

box-shadow : box에 그림자 효과를 주는 속성

선택자{box-shadow : 수평거리 수직거리 흐림 번짐 색}
