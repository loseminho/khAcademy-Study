BOM

window

window.open : 높이와 너비를 입력해서 새 창을 띄움

window.setTimeout : 일정 시간 후, 함수를 한번 실행

window.clearTimeout : setTimeout을 종료

window.setInterval : 일정 시간마다 함수를 반속 실행

window.clearInterval : setInterval을 종료

window.onload : 화면에 요소들이 로드 완료되면 자동으로 실행되는 함수

screen : 클라이언트 운영체제 화면 속의 속성 값을 갖는 객체

location : 브라우저 주소표시줄(URL)과 관련된 객체

navigator : 브라우저에 대한 정보
